Extenda OpenSource projects

Overview

This page contains an overview of all OpenSouce projects hosted on GitHub. They are all licensed under Apache License 2.0. Please use them, fork them and/or provide feedback in order to improve them

Extenda is an international IT company offering solutions for retail chains including department stores, fashion, food, specialty retail and general merchandise chains.

For more information please visit http://www.extenda.com

All projects are built using BuildHive provided by Cloudbees - see https://buildhive.cloudbees.com/job/extenda/

spdx-maven-plugin

spdx-maven-plugin is a maven plugin to generate a SPDX v 1.2 report file.

Fore more information visit http://extenda.github.io/spdx-maven-plugin

The project is hosted on GitHub https://github.com/extenda/spdx-maven-plugin
